# 집중 타이머 (Focus Timer)

시험기간에 쓰는 1-page 집중 타이머. 화면 가운데에 커다란 눈 한 쌍이 나를 지켜본다.

## 기능

### 눈
- 커서/터치 위치를 따라 눈동자 이동
- 모바일에서는 **자이로 센서**(기기 기울임)로 추적 — iOS는 첫 탭 시 권한 요청
- 유휴 상태에선 천천히 주변을 둘러봄
- 10초마다 깜빡임

### 타이머
- `HH:MM:SS 동안 집중함` 포맷
- **탭/앱이 숨겨지면 자동 정지** — 딴짓은 집중 시간으로 쳐주지 않음

### 이탈 페널티
- 탭/앱 전환 시 `visibilitychange` 감지
- 돌아오면 눈이 빨갛게 플래시 + 흔들림
- 풀스크린 빨간 오버레이: "어딜 갔다 온 거야?" + 자리 비운 시간 + 끊긴 스트릭

### 누적 기록 (localStorage)
상단 stats bar에 표시:
- **오늘** — 당일 누적 집중 시간 (자정에 리셋)
- **최장** — 역대 최장 연속 집중 기록 (이탈 없이)
- **이탈** — 오늘 이탈 횟수

### 종료 경고 (시간대별 카피)
종료 버튼을 누르면 집중 시간에 맞춰 다른 문구가 뜸:
- `< 1분`: 아직 1분도 안 됐는데 벌써 나간다고?
- `< 5분`: 5분도 못 채웠잖아
- `< 15분`: 밖에 안 했는데 벌써 나간다고?
- `< 30분`: 이제 좀 붙는 타이밍이잖아
- `< 1시간`: 이나 쌓았는데 여기서 접는다고?
- `< 1시간 30분`: 1시간 넘게 버텼는데 아깝잖아
- `1시간 30분+`: 잘 버텼어. 진짜 나갈 거야?

### 기타
- **Wake Lock API** — 공부 중 화면 자동 꺼짐 방지
- **모바일 최적화** — safe-area, 줌 차단, 홈 화면 추가 시 풀스크린 앱처럼 동작

## 사용법

1. `index.html`을 브라우저로 열기
2. 폰에서 쓸 땐 Safari/Chrome으로 접속 후 **홈 화면에 추가** → 풀스크린 앱처럼 동작
3. 페이지 열리면 자동으로 타이머 시작

## 기술 스택

- Plain HTML + CSS + JavaScript (의존성 없음)
- Noto Sans KR (Google Fonts)
- 단일 파일, 빌드 과정 없음
